30 A função CriaJogo() é responsável pela criação da janela do jogo e pelos componentes audio-visuais.
31 Ela deve ser chamada antes de qualquer outra função da biblioetac PIG.h. Não é possível criar dois
32 jogos no mesmo progama. Logo, somente a primeira chamada terá efeito.
33 Parâmetros:
34 nome_Janela (entrada, passagem por referência*): indica o nome que a janela do jogo mostrará no seu título.
35 É utilizado o operador * apenas por se tratar de um parâmetro string. O valor do parâmetro não é alterado dentro da função.
36 cursor_Proprio (entrada, passagem por valor): indica se o jogo utilizará cursores (ponteiros) próprios para o mouse ou utilizará o cursor padrão.
37 ********************************/
38 void CriaJogo(char *nome_Janela,int cursor_Proprio=0)
39 {
40  if (jogo==NULL){
41  jogo = new CJogo(nome_Janela,cursor_Proprio);
42  CAssetLoader::Inicia();
43  if (cursor_Proprio!=0)
44  CMouse::Inicia(jogo->renderer);
45  CGerenciadorTimers::Inicia();
46  CGerenciadorObjetos::Inicia(jogo->renderer);
47  CGerenciadorParticulas::Inicia(jogo->renderer);
48  CGerenciadorAnimacoes::Inicia(jogo->renderer);
49  }
50 }
